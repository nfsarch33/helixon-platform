--- helixon --help ---
helixon manages a Helixon agent runtime: validate config (doctor),
run the lifecycle (serve), and exercise tool dispatch interactively (repl).

Usage:
  helixon [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  doctor      Validate config and report runtime readiness
  help        Help about any command
  platform    Run the Helixon platform HTTP/SSE server (default :8787)
  repl        Interactive agent loop with tool dispatch (Ctrl-D to exit)
  serve       Start the Helixon runtime
  task        Execute a SprintBoard ticket using the agent loop
  version     Print binary version

Flags:
  -h, --help   help for helixon

Use "helixon [command] --help" for more information about a command.
